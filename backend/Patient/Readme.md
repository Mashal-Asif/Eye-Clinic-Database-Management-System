#  Patient Module – EyeClinicDB

## Overview
The Patient Module is a core part of the EyeClinicDB system.  
It is responsible for storing and managing all patient-related information such as personal details, contact information, and registration data.

This module is linked to the Authentication Module through the Users table, allowing optional user-based access for patients.

---

##  Purpose of This Module
- Store patient demographic information
- Maintain unique identification for each patient
- Link patients with system users (optional login access)
- Support clinic operations like appointments, prescriptions, and examinations
